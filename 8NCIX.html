① 志願理由
様式２・表面の主欄。アドミッション・ポリシー「求める学生像」（教育への情熱・探究心・子どもの成長を支える意欲）に沿って詳しく記入します。目安：手書きA4表面いっぱい（600〜800字程度）
A
小学校担任の先生への憧れを起点にした成長ストーリー型
「寄り添ってくれた恩師」という原体験を起点に、教師への志を自然な流れで語る
📌 この型が有効な場面 具体的なエピソードで読み手を引き込みたいとき。恩師体験という「証拠」を核に、子どもへの関心・自己成長・大学での学びを一本の線で繋げる。書類審査では具体性が最も評価されやすい。

私が福井大学教育学部初等教育コースを志願する理由は、小学校の担任の先生との出会いにあります。小学生のころ、私は学校に行くことが苦しくなった時期がありました。その時、担任の先生は毎日根気強く寄り添い、たくさんの言葉をかけてくれました。そのおかげで少しずつ前を向くことができ、「自分も将来、困っている子どもに寄り添える先生になりたい」と思うようになりました。

高校では、その思いをより深めるために英語の学習に力を入れました。将来の社会で必要になると考えたこともありますが、語彙を着実に増やし、英作文で実践的に使えるようになるという経験を通して、「積み重ねることで力がつく」という学びの喜びを実感しました。この体験は、子ども一人ひとりの小さな進歩を丁寧に支えたいという気持ちをさらに強くしてくれました。

また、探究活動では、大人の意見と異なると感じた場面で自分の考えを伝えることに苦労しました。根拠を集め、論理的に話す力を意識して鍛えた結果、周りに流されずに意見を発信できるようになりました。この経験から、子どもたちが主体的に意見を持ち、表現できる環境をつくることが教育の重要な役割だと感じています。

貴学初等教育コースは、幼小連携教育や地域と連携した実践的な教員養成を強みとしており、1年次から授業観察ができる点に大きな魅力を感じています。幼稚園から小学校への移行期を丁寧に支えるカリキュラムを学ぶことで、子ども一人ひとりの個性を引き出す小学校教員になるという目標に近づけると確信しています。入学後は地球連携カリキュラム研究にも取り組み、地域の教育課題に積極的に関わっていきたいと考えています。

B
探究活動での自己変革を軸にした意志・学術型
「意見を言えなかった」という弱点の克服を核に、教育への知的探究心を前面に出す構成
📌 この型が有効な場面 課題・困難の克服プロセスを示したいとき。「主体性・探究心」を重視する福井大学のAPと正面から呼応する。面接で「高校時代に頑張ったこと」と組み合わせて語りやすい。

私が福井大学教育学部初等教育コースを志願する理由は、探究活動での経験を通じて「主体的に学ぶことの大切さ」を痛感し、その力を子どもたちに届けたいと思うようになったからです。

高校の探究活動において、大人の方々の意見と異なる考えを持った場面で、自分の意見をうまく伝えられずに悩みました。そこで、事前に情報を集め、根拠を明確にして発言する習慣を意識的につくるようにしました。その結果、周囲に流されることなく自信をもって自分の考えを言えるようになりました。この経験が、子どもたち一人ひとりが「自分の考えを大切にして発信できる力」を育てることへの強い関心につながっています。

また、小学校時代の担任の先生が、登校が苦しかった私に毎日寄り添い、励まし続けてくれた経験も、教師を目指す原点となっています。教師という仕事は、子どもの学力を伸ばすだけでなく、その子の心に近づき、人生の中で「前を向くきっかけ」を与えられる職業だと感じています。

貴学は教育養成フラッグシップ大学として選ばれており、実践的な指導力が身につく環境が整っています。特に、リゾン型学びのネットワーク（異なる学年で学び合い、複数の科目を橋渡しする学習形態）と1年次からの授業参観制度は、現場に即した学びを積み上げるために最適な環境です。入学後は地球連携カリキュラム研究を中心に学び、地域の学校と連携しながら、子どもたちの主体的な学びを支える教育を探究していきたいと考えています。

C
幼小連携・地域教育ビジョンを前面に出す目的意識型
「なぜ福井大学でなければならないか」を最も明快に示す。大学の特色との整合性が高い構成
📌 この型が有効な場面 「なぜこの大学・このコースでなければならないのか」を強調したいとき。貴学固有の特色（幼小連携、地球連携、STEAMラボ等）と志望理由を直結させるため、書類審査で差別化しやすい。

私が福井大学教育学部初等教育コースを志願する理由は、幼稚園から小学校への移行期に着目した教育に強く惹かれているからです。私自身、幼稚園から小学生になったとき、何もかもが一気に変わる環境の変化に戸惑い、学校が苦しい場所になった時期がありました。その時、小学校の担任の先生に毎日寄り添っていただき、立ち直ることができました。この体験から、入学直後のこの「移行期」を丁寧に支える教師の存在がいかに重要かを身をもって感じています。

福井大学初等教育コースの2系列は、幼稚園と小学校をつなぐことを学ぶために設計されており、私の問題意識と深く一致しています。また、リゾン型学びのネットワーク、STEAMラボ、地域と連携した実践的な教職演習など、時代の変化に対応した最新の学習環境が整っている点も他大学にはない大きな魅力です。1年次から授業参観ができることで、理論と現場を早い段階から結びつけながら学べると考えています。

高校の探究活動では、大人と異なる意見を持つ場面で根拠を集めて発言する習慣を身につけ、周囲に流されない自分の意見を表現できるようになりました。この「主体的に考え、伝える力」は、子どもたちにも育んでいきたい力そのものです。

入学後は、地球連携カリキュラム研究に取り組み、1年次に学校現場を実際に見て、2〜3年次には合宿等を通じて「学校」と「地域側」の視点を深めていきたいと考えています。将来は福井で小学校の先生となり、子ども一人ひとりの個性を引き出しながら、多様な背景を持つ子どもたちに寄り添える教師を目指します。

② 将来の進路希望（校種や教科を具体的に）
裏面の記入項目①。校種・教科・地域など具体的な記述が求められます。3〜5行程度の記述が適切です。
A
端的・シンプル型
明快さを重視。他の欄との重複を避けてすっきり書く
📌 この型が有効な場面志願理由の欄に十分な情報を書いた場合。重複を避け、端的に伝える。
　福井県内の公立小学校の教員を目指しています。特に、低学年の担任として子ども一人ひとりに寄り添いながら、学校生活のスタートを支えていきたいと考えています。将来的には学級担任として全教科を指導しながら、子どもの主体的な学びを引き出す授業づくりに取り組んでいきたいです。
B
理由付き・丁寧型
なぜその校種・地域なのかの理由を添える
📌 この型が有効な場面地元貢献の意欲やコース選択の必然性を合わせてアピールしたいとき。
　福井県内の公立小学校の教員を目指しています。自分自身が地元の小学校で担任の先生に支えられた経験から、地元の子どもたちに貢献したいという思いが強くあります。学級担任として全教科を指導することを基本としながら、特に国語や道徳の時間を通じて、子どもたちが自分の考えを言葉で表現する力を育てていきたいと考えています。また、特別支援の視点も大切にし、様々な背景を持つ子どもたちに寄り添える教員を目指したいです。
C
段階・ビジョン型
入学後〜卒業後の段階的なビジョンを示す
📌 この型が有効な場面長期的な目標の明確さをアピールしたいとき。「主体性・目的意識」を評価するAPに合致する。
　卒業後は福井県内の公立小学校の教員になることを第一の目標としています。将来は低学年の担任として、幼小連携の視点を活かしながら、子どもが安心して学べる学級環境をつくりたいと考えています。さらに経験を積んだ後は、学年主任や特別支援コーディネーターなど、学校全体の教育体制を支える立場にも挑戦し、地域と連携した学校づくりに貢献していきたいです。
③ 自分が特に優れていると思っていること
裏面の記入項目②。強みをエピソードで裏付けます。3〜5行程度。
A
負けず嫌い・継続力型
試験の成績が不振だった経験から学習計画を立て直した具体的エピソードで強みを示す
📌 この型が有効な場面「学習の自発性・計画性・持続性」を問う推薦書・面接との整合性を高めたいとき。
　自分が特に優れていると感じる点は、諦めずに粘り強く取り組む継続力です。模擬試験の成績が思い通りに出なかった際、落ち込むだけでなく、すぐに勉強計画を見直し、積極的にわからない部分を先生に質問しに行きました。どんなに難しい課題に直面しても最後まで諦めずに取り組む姿勢は、子どもたちへの粘り強い支援にも活かせると考えています。
B
主体的発信力型
探究活動での根拠を持った発言の習慣化を強みとして示す
📌 この型が有効な場面APの「主体性・協働的に学ぶ態度」に直結させたいとき。志願理由（パターンB）との一貫性が高い。
　自分が特に優れていると感じる点は、根拠を持って自分の意見を発信できる力です。高校の探究活動において、大人の意見と異なると感じた場面でも、事前に情報を収集し、根拠を明確にしてから発言する習慣を意識的に身につけました。その結果、周囲の雰囲気に流されずに自信をもって意見を述べられるようになりました。この力を、子どもたちが自分の考えを大切にする授業づくりに活かしていきたいです。
C
明朗・人間関係構築力型
「常に明朗に人と接することができる」という強みをエピソードで具体化する
📌 この型が有効な場面子どもとの関係構築や「コミュニケーション能力」を重視する面接対策と合わせたいとき。
　自分が特に優れていると感じる点は、常に明るく人と接することができることです。初対面の人に対しても自然に笑顔で話しかけることができ、放課後の子どもとの関わりやボランティア活動の場でも、子どもたちがすぐに打ち解けてくれると感じています。また、どんな状況でも最後まで諦めずに物事に取り組む粘り強さも持ち合わせており、信頼関係を築きながら地道に努力を続けることが自分の強みだと考えています。
④ 好きな教科・科目とその理由
裏面の記入項目③。英語への取り組みを中心に、教育との接続も意識して記述します。3〜4行程度。
A
英語・継続学習軸
「毎日続けた」という継続力を英語への取り組みで示す
📌 この型が有効な場面「勉強の自発性・計画性・持続性」を英語学習で示したいとき。
　最も力を入れてきた科目は英語です。将来の社会でも必ず役立つ力だという確信から、高校3年間を通じて毎日欠かさず単語学習を続けました。継続する中で語彙が増えるだけでなく、覚えた単語を英作文の中で実際に使えるようになるという実践的な力が身についたと感じています。また、新聞の英語記事や英語のテキストを読む習慣が読解力を高めてくれました。この「毎日続ける力」を教師としての学び続ける姿勢にも活かしていきたいです。
B
英語・実践力軸
英語力の実践的な使い方と、小学校英語教育への関心を接続する
📌 この型が有効な場面小学校英語指導への関心を将来の教職と結びつけたいとき。
　最も好きな科目は英語です。単語を覚えるだけでなく、英作文の中で実際に使えるかを意識しながら学習を進めてきました。毎日継続して語彙を増やした結果、文章の読み書きに自信がつき、英語で考えることへの楽しさを感じられるようになりました。小学校でも英語教育の重要性が高まっている現在、子どもたちに英語を「楽しいもの」と感じてもらえるような指導ができる教員になりたいという気持ちも高まっています。
C
英語＋国語・複合軸
英語と国語の両方への関心を示すことで、言語教育への幅広い意欲をアピールする
📌 この型が有効な場面言語系全般への関心の広さを示したいとき。「表現する力」を育てる教師像と連携させやすい。
　特に力を入れてきた科目は英語と国語です。英語は毎日の単語学習と英作文練習を通じて、実践的な運用力を身につけてきました。国語については、文章を読んで自分の考えをまとめる力が、すべての教科学習の根幹になると感じており、論述問題に積極的に取り組んできました。言葉を正確に理解し、自分の考えを適切に表現する力は、子どもたちにも大切にしてほしい能力であり、言語教育の面から子どもの学びを支えていきたいと考えています。
⑤ 趣味・特技・有している資格
裏面の記入項目④。簡潔に、かつ教育との接続が見えるように書くと効果的です。2〜3行程度。
A
端的列挙型
📌 この型が有効な場面他の欄が充実しているので、この欄は簡潔にまとめたいとき。
　趣味は子どもと関わること（放課後の子どもたちとの交流、地域ボランティア）、英語の学習、読書です。特技は継続して物事に取り組むことで、3年間毎日の英単語学習を続けました。現在、英検2級を取得しており、準1級取得に向けて学習を続けています。
B
意義付き型
📌 この型が有効な場面趣味・特技の記述に人間的な厚みを加えたいとき。
　趣味は子どもと関わることで、放課後に地域の子どもたちと交流する機会を大切にしています。子どもの言葉や表情から感情を読み取る感性が、自然と磨かれてきたと感じています。特技は継続力で、英語の毎日学習を3年間休まず続けました。保有資格は英検2級で、さらに上位の資格取得を目指しています。
C
教育接続型
📌 この型が有効な場面趣味・特技と教師としての資質を明確に結びつけたいとき。
　趣味は子どもと関わることと読書です。子どもと接することで、一人ひとりの個性の違いに自然と興味を持つようになり、教育への関心がさらに高まりました。特技は粘り強く継続する力です。3年間毎日の英語学習を続けてきた経験が、この自信の根拠になっています。英検2級を保有しており、今後も資格取得を通じて自己研鑽を続けていきたいと考えています。
